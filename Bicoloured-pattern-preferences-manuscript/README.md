# Bicoloured-pattern-preferences-manuscript
Repository containing data for the manuscript titled "Spontaneous preferences of free-flying honeybees for mono-coloured or multi-coloured flower patterns"

This readme file describes the data files accompanying the above publication. For any further
queries please contact scarlett.howard@monash.edu

The following file is included:
Sheet 1) “beedataYQ.csv”
This file contains the individual choices made by insects during the training. The columns are
as follows:
BEEID: ID of individual insect
PREFERENCE: experimental condition
CHOICE: binary choice data
TRIAL: trial number

Sheet 2)
This sheet contains the key for sheet 1
