# Symmetry_and_quantity_bee_manuscript
 Repository containing data for our manuscript titled 'Working title'

Visit here for html page: https://tcleslieng.github.io/Symmetry_and_quantity_bee_manuscript/

1) "sym_test.csv"

This file contains individual choices made by bees in the test phase. The columns are as follows:

* BEEID: individual bee recruited from the hive
* CNUM: choice number
* CHOICE: binomial choice made by bees
* LOWNUM: of the pair of quantities shown, this was the lower number
* HIGHNUM: of the pair of quantities shown, this was the higher number
* TESTCONDITION: test condition
* TEST ORDER: counterbalancing order of tests
* TRAINING CONDITION: training condition allocated to the bee

2) "sym_training.csv"

This file contains individual choices made by bees in the training condition. The columns are as follows:

* BEEID: individual bee recruited from the hive
* CNUM: choice number
* CHOICE: binomial choice made by bees
* SYM: training condition, S = symmetry A = assymetry
* LOWNUM: of the pair of quantities shown, this was the lower number
* HIGHNUM: of the pair of quantities shown, this was the higher number
* SHAPE: shape of stimuli presented
